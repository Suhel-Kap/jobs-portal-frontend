# How to setup and run this project

1. Clone the repository

```bash
git clone https://github.com/Suhel-Kap/jobs-portal-frontend.git
```

2. Change the working directory

```bash
cd jobs-portal-frontend
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

5. Open the app

```bash
http://localhost:3000
```
